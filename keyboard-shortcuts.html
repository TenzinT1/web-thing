Accessing Site Via Keyboard

Keyboard Access Instructions

This website fully supports keyboard navigation. You can use the following keys:

  • Tab - Allows user to move forward through navigation links and page content
  • Shift + Tab - Alows user to move backward
  • Enter - Activate a link or button
  • Arrow Keys - Allows user to scroll through pages

Keyboard Support Goals

I made sure for this part of the assignment that the following goals were met:

  • All navigation items are reachable using the Tab key
  • All links on the “Things to Read” page can be accessed via the keyboard
  • Navigation buttons have indicators that are visible and easy to follow
  • The navigation bar appears consistently on all pages
